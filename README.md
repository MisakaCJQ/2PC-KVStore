# 2PC-KVStore

基于两阶段协议实现的简单分布式KV存储

是在学校的时候写的云计算课程实验

[GitHub - 1989chenguo/CloudComputingLabs](https://github.com/1989chenguo/CloudComputingLabs)



## 代码运行方法

```bash
make
./kvstore2pcsystem --config_path [config_path]
```
